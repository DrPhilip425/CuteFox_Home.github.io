用Hexo+Github搭建一个自己的博客

2023/8/5 教程 仅适用于Windows Hexo

1.准备搭建环境

(1)安装适用于Windows系统的Git
https://git-scm.com/downloads
(2)安装适用于Windows系统的NodeJs环境
https://nodejs.org/zh-cn/download

2.获取SSH公钥

(1)创建github账号
https://github.com/
(2)获取本地密钥

(3)将SSH公钥绑定gitHub

3.创建网页

(1)Windows使用PowerShell
(2)安装配置
npm install -g hexo-cli
(3)绕过执行策略的限制
powershell -ExecutionPolicy Bypass
(4)创建文件夹package(package为文件夹名可修改)
hexo init package
(5)进入文件夹package
cd package
(6)再次安装
npm install

4.部署服务器

(1)再次绕过执行策略的限制
powershell -ExecutionPolicy Bypass
(2)生成静态文件
hexo g
(3)运行本地服务器
hexo s

5.部署网站

(1)新建一篇文章title(title可修改成想要的标题)
hexo new "title"
(2)保存在本地
npm install hexo-deployer-git --save
(3)检查
npm list hexo-deployer-git
(4)再次绕过执行策略的限制
powershell -ExecutionPolicy Bypass
(5)部署网站
hexo d

参考资料：https://hexo.io/zh-cn/docs/
